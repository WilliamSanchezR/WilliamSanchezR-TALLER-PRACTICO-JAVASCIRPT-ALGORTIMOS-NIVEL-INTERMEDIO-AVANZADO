Ejercicio condicional #1: Sistema de Calificaciones Detallado

Estructura condicional

Un colegio necesita un sistema para evaluar el rendimiento de los estudiantes en una materia específica. El sistema debe permitir ingresar 10 notas numéricas (entre 0 y 5). Luego, calculará el promedio de estas notas y determinará si el estudiante aprobó o no (se considera aprobado con un promedio de 3.0 o superior). Finalmente, mostrará la calificación literal correspondiente al promedio, utilizando la siguiente escala:
● 0.0 a 2.9: "Reprobado"
● 3.0 a 3.9: "Aceptable"
● 4.0 a 4.4: "Sobresaliente"
● 4.5 a 5.0: "Excelente"
Salida Esperada: El sistema mostrará el promedio numérico de las 10 notas, un mensaje indicando si el estudiante aprobó o reprobó, y la calificación literal correspondiente.

Análisis:

A continuación se describe el análisis del algoritmo

Datos de entrada:

  • Primer Dato: Notas numéricas del estudiante (se ingresan 10).

    • Proceso:

    Declaramos una variable la cual se usara para sumar todas las notas, usamos un bucle que repetira 10 veces para que el usuario pueda ingresar las 10 notas, pero antes de aceptar la nota se validara si el valor ingresado esta entre 0 y 5, si es valida se sumara al total y continuara si no es valida arrojara error y no avanzara mas, luego de calcula el promedio dividiendo la suma total de notas con 10, dependiendo del resultado nos arrojara el mensaje que se encuentra en la condicional basada en el promedio, y se determina si el estudiante aprobo o no. y al final se mostrara el mensaje con el resultado y si aprobo o no.

    Datos o mensajes de salida:

    - Promedio final del estudiante (ejemplo: 3.7).
    - El estado del estudiante si aprobó o reprobó.
    - Calificación literal: "Excelente", "Sobresaliente", "Aceptable" o "Reprobado"
    - Por ultimo el mensaje de salida final con la concatenacion total de todo.

    Función relizada: 

                    
                    function rendimientoEstudiante () {

                    alert("Algoritmo para evaluar el rendimiento de los estudiantes en una materia especifica el sistema debe permitir ingresar 10 notas numericas entre 0 y 5 luego calculara el promedio de estas notas y determinara si el estudiante aprobo o no y se considera aprobado con un promedio de 3.0 o superior finalmente debera mostrar la calificacion literal correspondiente al promedio");

                    let total = 0;

                    // Usamos un bucle for que se repite 10 veces (una por cada nota) use esto por lo que el instructor explico xd
                    for (let i = 0; i < 10; ) {
                        let nota = parseFloat(prompt("Ingresa la nota número " + (i + 1))); // Pedimos la nota al usuario
                        if (nota >= 0 && nota <= 5) {                                       // Verificamos que la nota sea válida (entre 0 y 5)
                            total = total + nota; // sumamos la nota al total, esto lo hacemos gracias a que cada vez que agreguemos una nota se sumara a la anterior que se agrego
                            i++; // solo aumentamos si la nota fue válida
                        } else {
                            alert("Nota inválida. Debe estar entre 0 y 5."); // mensaje de error
                        }
                    }

                    let promedio = total / 10;
                    let mensaje = "";
                    let resultado = "";

                    // Según el promedio, asignamos el mensaje que sacara el estudiante por la nota obtenida
                    if (promedio >= 4.5) {
                        mensaje = "Excelente";
                    } else if (promedio >= 4.0) {
                        mensaje = "Sobresaliente";
                    } else if (promedio >= 3.0) {
                        mensaje = "Aceptable";
                    } else {
                        mensaje = "Reprobado";
                    }

                    // Decidimos si aprobó o no
                    if (promedio >= 3.0) {
                        resultado = "aprobó";
                    } else {
                        resultado = "reprobó";
                    }

                    // Mostramos el resultado final al usuario
                    alert("El estudiante " + resultado + " con un promedio de " + promedio + " - " + mensaje);
                }
